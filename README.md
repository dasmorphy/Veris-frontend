Necesario ejecutar npm i -f
Ejecutar npm run dev para levantar el proyecto
Levantar el back de node alojado en: https://github.com/dasmorphy/Veris

Documentacion: 
-src/app/ – código principal de la aplicación.

-components/ – componentes de interfaz de usuario reutilizables.

-pages/ – pantallas/páginas principales del sistema.

-services/ – servicios de datos y comunicación API.

-guards/ – control de acceso a rutas.

-models/ – interfaces y tipos de datos.

-assets/ – imágenes, estilos y archivos estáticos.

-environments/ – configuración específica del entorno.
